# **App Name**: Amor Digital

## Core Features:

- Headline Display: Display a catchy headline to capture user attention upon landing on the page.
- VSL Integration: Embed a Video Sales Letter (VSL) in a 16:9 aspect ratio to showcase the product.
- User Data Input: Capture basic user data via a form to initiate engagement and personalize the experience.

## Style Guidelines:

- Primary color: Warm Coral (#FF7F50) to evoke feelings of warmth and connection.
- Background color: Light Beige (#F5F5DC), offering a soft and unobtrusive backdrop.
- Accent color: Muted Teal (#70DBD4) for highlights and calls to action, ensuring contrast without being jarring.
- Headline font: 'Playfair', serif font, for a sophisticated and trustworthy feel; body text: 'PT Sans' sans-serif.
- Clean and responsive layout to ensure optimal viewing on all devices, emphasizing the VSL section.
- Subtle fade-in effects on page load and content appearance to enhance user experience without being distracting.
